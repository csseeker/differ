# Release Notes - v0.0.2-alpha

**Release Date:** October 1, 2025  
**Tag:** `v0.0.2-alpha`

## Summary

This release brings significant quality-of-life improvements including **professional structured logging** for better diagnostics, **enhanced user experience** with a new About dialog, and **unified icon management**. The application now provides better error reporting, improved branding consistency, and comprehensive test coverage for reliability.

## üéØ New Features

### Structured Logging & Diagnostics
- **Professional Logging**: Comprehensive logging with file, console, and debug output
- **Configuration-Based**: Manage log levels via `appsettings.json` without code changes
- **Environment-Specific Settings**: Separate Development and Production logging configurations
- **Runtime Control**: Dynamic log level adjustments via `IDifferLogLevelManager` service
- **Automatic File Logging**: Logs saved to `%LOCALAPPDATA%\Differ\Logs` with automatic rotation
- **Startup Error Capture**: Fallback mechanisms to diagnose critical startup failures
- **Strongly-Typed Configuration**: Validated logging options for reliability

### Enhanced User Experience
- **About Dialog**: View version information, copyright, and GitHub repository link
- **Help Menu**: Easy access to "About Differ" from the main menu
- **Improved Dialogs**: Consistent dialog management throughout the application
- **Thread-Safe Operations**: Responsive UI with async dialog support

### Application Icons
- **Professional Icons**: High-quality application icon in title bar, taskbar, and Alt+Tab
- **Windows Integration**: Proper icons for MSIX package and Start menu
- **Consistent Branding**: Unified icon assets across all Windows touchpoints

### Quality & Reliability
- **Comprehensive Test Coverage**: New unit tests for core comparison engine
- **Model Testing**: Validation for `ComparisonItem`, `DirectoryComparisonResult`, and `FileSystemItem`
- **Service Testing**: Coverage for directory scanning, file comparison, and text diff operations
- **Edge Case Handling**: Tests for empty files, large files, binary files, and error conditions

## üì¶ Installation

### Portable Version (Recommended)

1. Download `Differ-portable-win-x64.zip` (66.8 MB) from the release assets
2. Extract to any writable folder on your system
3. Run `Differ.App.exe`
4. Windows SmartScreen may prompt you to confirm the first time - click "More info" ‚Üí "Run anyway"

### MSIX Package (Windows Store Integration)

1. Download `Differ_0.0.2.0_x64.msix` (1.5 MB) from the release assets
2. Double-click to install (requires developer mode or certificate installation)
3. Provides Start menu integration and automatic updates
4. See `docs/MSIX_PACKAGING.md` for detailed installation instructions

## üîß System Requirements

- **OS**: Windows 10 (1809+) or Windows 11
- **Architecture**: x64
- **.NET**: .NET 8.0 Runtime (included in portable version)
- **Disk Space**: ~70 MB for portable version
